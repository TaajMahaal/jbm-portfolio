# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-26

### Added
- Initial release of retro-styled portfolio
- Interactive terminal with easter eggs
- Draggable window system
- Boot sequence animation
- Matrix-style background effect
- CRT screen effects (scanlines, glow, vignette)
- Konami Code easter egg
- Comprehensive CV content
- Mobile-responsive design
- TypeScript strict mode
- Tailwind CSS v4 integration
- Framer Motion animations
- Three.js background effects

### Features
- **Terminal Commands**: help, whoami, history, ls, cat, neofetch, git log, and more
- **Easter Eggs**: Hidden commands, Konami Code, source code message
- **UI/UX**: Retro OS interface with modern polish
- **Performance**: Optimized Astro build with island architecture
- **Accessibility**: Semantic HTML and ARIA labels

### Tech Stack
- Astro 5.x
- React 19.x
- TypeScript (strict mode)
- Tailwind CSS v4
- Framer Motion
- Three.js / React Three Fiber
